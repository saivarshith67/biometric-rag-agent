# Firmware Version 1.0.3 (Build No. 1.0.3_210126)

Release: 2021-02-02

# New Features and Improvements

1. Changes to a new LCD screen
- After upgrading a firmware to v1.0.3, you cannot downgrade the firmware to the previous version that includes the kernel and U-Boot.

# Bug Fixes

1. When a user authenticated using the iCLASS Seos AoC (Access on Card) after saving a face in it, the device timed out or recognized the AoC as a CSN.
2. Motion detection did not work properly after scanning a card.
3. When using T&A mode, the device did not recognize a card intermittently.
