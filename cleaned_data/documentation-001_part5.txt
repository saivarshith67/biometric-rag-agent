# Getting Started

# Cables and connectors

# Power

| Pin | Name     | Color                |
| --- | -------- | -------------------- |
| 1   | PWR +VDC | Red (White stripe)   |
| 2   | PWR GND  | Black (White stripe) |

# Relay

| Pin | Name    | Color  |
| --- | ------- | ------ |
| 1   | RLY NO  | White  |
| 2   | RLY COM | Blue   |
| 3   | RLY NC  | Orange |

| Pin | Name     | Color  |
| --- | -------- | ------ |
| 1   | 485 TRXP | Blue   |
| 2   | 485 TRXN | Yellow |
| 3   | 485 GND  | Black  |
| 4   | SH GND   | Gray   |
