# BioStar 2 Server and Device Port Usage

For device with LCDs, you can set it on the device as well.

| Type        | Ethernet       |
| ----------- | -------------- |
| Port        | 51211          |
| DHCP        | Enabled        |
| IP Address  | 192.168.16.141 |
| Gateway     | 192.168.16.1   |
| Subnet Mask | 255.255.255.0  |
