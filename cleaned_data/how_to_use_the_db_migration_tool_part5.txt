# How to use the DB migration tool

1. To start the migration, type run_user.bat and press Enter.
2. Copy biostar2.db from B2DM\db folder and paste it to C:\Program Files (x86)\BioStar 2\db folder.
3. Start the BioStar 2 Launcher Service from Services.
